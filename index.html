个人博客，顾名思义，就是个人技术栈的点滴积累，整理成文后，以备将来回顾往事时，博大家一笑。

博客对于我而言，不只是给代码找一个温馨的家，更多时候，是它在教我如何梳理知识体系。比如此博客 托管在 GitHub Pages 服务上，这是一个免费的建站服务，由全球最大的开源社区 GitHub 所提供。而此 服务又依赖 Jekyll 工具来编译仓库源文件，渲染为当前看到的网页。Jekyll 本身是独立的工具，但它 也支持使用 Markdown 编写纯文本内容，然后发布出来。而存储这个网页源代码用的是 Git 工具，这是 一种代码版本管理工具，非常好用，它会记录你全部的有效工作，可以随时查阅那些改动，以及还原到 之前的状态。这一系列服务和工具梳理下来，我发现我的思路已经无比畅通，所以写博客，真的很美妙。
